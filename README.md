# Telas do Administrador

## 1. Dashboard (Painel Principal)

Tela inicial com visão geral do sistema.

**Deve conter:**

* Total de imóveis cadastrados
* Total de reservas
* Reservas pendentes
* Reservas confirmadas
* Reservas canceladas
* Total de clientes
* Imóveis disponíveis / ocupados
* Gráficos (reservas por mês, imóveis mais reservados)

---

## 2. Gestão de Imóveis

### 2.1 Lista de Imóveis

**Funções:**

* Ver imóveis cadastrados
* Pesquisar imóveis
* Filtrar por status (Disponível / Reservado / Indisponível)
* Editar imóvel
* Excluir imóvel
* Ver detalhes

### 2.2 Cadastrar Imóvel

**Campos sugeridos:**

* Título do imóvel
* Tipo (Apartamento, Moradia, Escritório, etc.)
* Localização
* Preço
* Descrição
* Número de quartos
* Número de casas de banho
* Área
* Imagens do imóvel
* Status (Disponível / Indisponível)
